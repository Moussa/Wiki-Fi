About

Wiki-Fi is a tool that analyzes user activity on a wiki and produces graphs and metrics for display.

Currently supported wikis include the Official Team Fortress Wiki, the Portal Wiki, the Dota2 Wiki, and the Combine Overwiki.

Wiki-Fi makes use of the following software:

  • Python
  • MongoDB
  • Bootstrap
  • jQuery
  • Google Chart Tools
  • jQCloud
  • Heading Anchors
View the source code on GitHub.
